Supabase — Auth, DB & Storage

Créer le projet, lancer les migrations, configurer l'auth

~3 min
+
Démarrer Supabase en local

supabase db push
Insérer les données de seed (templates)
supabase db reset --linked
Générer les types TypeScript
pnpm db:types
